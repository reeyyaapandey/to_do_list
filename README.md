# to_do_list
An advanced To-Do List GUI application built using Python and Tkinter. It allows users to add, edit, delete, and mark tasks as completed, with tasks saved locally using file handling.
The Advanced To-Do List GUI Application is a desktop-based task management system developed using Python and Tkinter. This project is designed to help users efficiently organize their daily tasks through a simple and interactive graphical interface. It allows users to create, update, delete, and track tasks while ensuring data persistence using file handling.

This project demonstrates core Python concepts such as GUI programming, event handling, file operations, and clean code structure, making it ideal for beginners aiming to strengthen their practical skills.
